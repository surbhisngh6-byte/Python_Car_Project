# Used Car Price Analysis using Python & Pandas

## Objective
Analyze used car market data using Python and Pandas to identify pricing trends, vehicle characteristics, and business insights.

## Dataset
car_prices.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
1. Data Loading and Inspection
2. Missing Value Analysis
3. Duplicate Detection
4. Data Cleaning
5. Data Aggregation and Queries
6. Data Visualization
7. Business Insights

## Key Findings
- Average selling price: 13,611
- Most sold model: Nissan Altima
- Highest average selling price brand: Rolls-Royce
- Vehicle age and mileage negatively impact price
- Vehicle condition positively impacts price
